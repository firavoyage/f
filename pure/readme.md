对于成熟的且前人之述备矣的学科，新加入的教科书作者有一个回避不了的问题，那就是读者凭什么要在你的作品上花费时间和金钱呢？

这个问题徐一鸿回答过，Weinberg 也回答过。关于为什么要写自己的量子场论教科书，他们的回答是 “I wrote the book that I would have liked to learn from.” 他们是在形成自己的思想体系后，在清楚地知道前人作品的局限性的前提下，去尽力创作另一个时空中作为初学者的自己能够喜欢甚至偏爱的书。
这种动机，跟利用微不足道的知识差在懵懂的初学者群体中炫耀甚至吸金的动机相比，完全是云泥之别。

Maki‘s Lab 的作者扪心自问，你们真的喜欢自己写的东西吗？你们愿意当年作为初学者的自己用这些书去学习吗？
