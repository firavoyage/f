# 1

求二面角. 使用向量. 什么时候 sin = abs(cos) 什么时候 sin = sqrt(1-cos^2)

---

在计算二面角时，使用向量方法通常涉及两个平面的法向量。设两个平面的法向量分别为 \(\mathbf{n}\_1\) 和 \(\mathbf{n}\_2\)，则二面角 \(\phi\)（通常定义为 \(0^\circ\) 到 \(180^\circ\) 之间的角）的余弦和正弦可以通过以下公式计算：

- **余弦值**：
  \[
  \cos \phi = \frac{|\mathbf{n}\_1 \cdot \mathbf{n}\_2|}{|\mathbf{n}\_1| |\mathbf{n}\_2|}
  \]
  这里取绝对值的目的是确保二面角 \(\phi\) 为锐角或直角（即 \(0^\circ \leq \phi \leq 90^\circ\))，但如果您需要完整的 \(0^\circ\) 到 \(180^\circ\) 范围，可以不取绝对值，此时 \(\cos \phi\) 可能为负，表示钝角。

- **正弦值**：
  \[
  \sin \phi = \frac{|\mathbf{n}\_1 \times \mathbf{n}\_2|}{|\mathbf{n}\_1| |\mathbf{n}\_2|}
  \]
  这里叉积的模给出了二面角的正弦值，且总是非负的，因为二面角的正弦在 \(0^\circ\) 到 \(180^\circ\) 范围内非负。

### 关于 \(\sin \phi\) 与 \(\cos \phi\) 的关系

- **\(\sin \phi = \sqrt{1 - \cos^2 \phi}\)**：  
  这是三角恒等式，在二面角计算中总是成立，因为 \(\sin \phi \geq 0\) 对于 \(\phi \in [0^\circ, 180^\circ]\)。因此，如果您已经计算出 \(\cos \phi\)（无论是否取绝对值），都可以用这个公式计算 \(\sin \phi\)。但注意，当 \(\cos \phi\) 接近 ±1 时，数值计算可能不精确，此时直接使用叉积计算正弦值更可靠。

- **\(\sin \phi = |\cos \phi|\)**：  
  这仅在二面角 \(\phi = 45^\circ\) 或 \(135^\circ\) 时成立，因为此时 \(\sin \phi = |\cos \phi| = \frac{\sqrt{2}}{2}\)。在一般情况下的二面角计算中，这个等式不成立，因此不应常规使用。

### 总结

- 在向量方法中，二面角的正弦值通常通过叉积直接计算，或通过点积得到的余弦值使用 \(\sin \phi = \sqrt{1 - \cos^2 \phi}\) 计算。
- \(\sin \phi = |\cos \phi|\) 仅适用于特定角度（45° 或 135°），不是一般情况。

# 2

高中数学没有叉积

---

好的，完全理解。在高中数学的立体几何中，使用向量法求二面角时，我们通常只使用点积，而不使用叉积（因为这是大学内容）。

我们来清晰地梳理一下在高中数学框架下，如何使用向量法求解二面角，并解答您关于正弦和余弦公式的疑问。

### 第一步：求二面角的平面角

这是最关键的一步。二面角的大小是用它的**平面角**来度量的。

1.  在两个平面的交线上找一个点。
2.  在这个点上，分别在两个平面内作垂直于交线的直线。
3.  这两条垂线所成的角，就是二面角的平面角，记作 \(\theta\)。\(\theta\) 的范围通常是 \([0^\circ, 180^\circ]\)。

### 第二步：使用法向量求平面角的余弦

设平面 \(\alpha\) 和 \(\beta\) 的法向量分别为 \(\vec{n_1}\) 和 \(\vec{n_2}\)。

**核心关系：** 两个法向量的夹角 \(\langle \vec{n_1}, \vec{n_2} \rangle\) **等于** 二面角的平面角 \(\theta\) **或其补角**（即 \(180^\circ - \theta\)）。

这是因为法向量与平面垂直，它们之间的关系存在两种可能的方向。

所以，我们直接计算法向量夹角的余弦：
\[
\cos \langle \vec{n_1}, \vec{n_2} \rangle = \frac{\vec{n_1} \cdot \vec{n_2}}{|\vec{n_1}| |\vec{n_2}|}
\]

**那么，这个值到底是 \(\cos \theta\) 还是 \(\cos (180^\circ - \theta) = -\cos \theta\) 呢？**

**高中数学的标准处理方法：**

**观察法！** 根据你所建立的空间直角坐标系和所求二面角的位置，**直观判断**这个二面角是锐角还是钝角。

- **如果你判断二面角是锐角**：那么取计算结果的**绝对值**。
  \[
  \cos \theta = \frac{|\vec{n_1} \cdot \vec{n_2}|}{|\vec{n_1}| |\vec{n_2}|}
  \]
  这样可以保证结果是非负的，对应锐角。

- **如果你判断二面角是钝角**：或者题目要求求出具体的角度值（而不仅仅是正弦值），那么就直接使用计算值（可能是负的）。
  \[
  \cos \theta = \frac{\vec{n_1} \cdot \vec{n_2}}{|\vec{n_1}| |\vec{n_2}|}
  \]
  如果结果为负，说明 \(\theta > 90^\circ\)。

**如果无法直观判断，通常默认二面角为锐角，使用绝对值公式。这在很多小题中是默认规则。**

---

### 第三步：从余弦求正弦（回答您的核心问题）

现在我们有了二面角平面角 \(\theta\) 的余弦值 \(\cos \theta\)，如何求其正弦 \(\sin \theta\)？

**1. 什么时候用 \(\sin \theta = \sqrt{1 - \cos^2 \theta}\) ？**

- **这是通用且最安全的方法。**
- 因为二面角 \(\theta\) 的范围是 \([0^\circ, 180^\circ]\)，在这个范围内，\(\sin \theta\) **永远是非负的**（\(\sin 0^\circ = 0, \sin 90^\circ = 1, \sin 180^\circ = 0\)）。
- 所以，无论你上一步的 \(\cos \theta\) 是正还是负，当你用这个公式时，**开根号后直接取正号**即可。
  \[
  \sin \theta = \sqrt{1 - \cos^2 \theta}
  \]
  **在高中数学的二面角计算中，请始终使用这个公式。**

**2. 什么时候用 \(\sin \theta = |\cos \theta|\) ？**

- **这几乎从来不是求二面角正弦值的正确公式。**
- 这个等式成立的条件是 \(\sin \theta = |\cos \theta|\)，解得 \(\tan \theta = 1\)，即 \(\theta = 45^\circ\) 或 \(135^\circ\)。
- 这只在二面角恰好是 \(45^\circ\) 或 \(135^\circ\) 这个**特例**下成立。把它当作一个普遍的公式来使用是**错误的**。

### 总结与例题步骤

**求解二面角 \(\theta\) 的步骤：**

1.  建立坐标系，求出两个平面的法向量 \(\vec{n_1}\) 和 \(\vec{n_2}\)。
2.  计算法向量夹角的余弦：\(\cos \langle \vec{n_1}, \vec{n_2} \rangle = \frac{\vec{n_1} \cdot \vec{n_2}}{|\vec{n_1}| |\vec{n_2}|}\)。
3.  **判断二面角的锐钝性：**
    - 若为锐角（或无法判断）：\(\cos \theta = \frac{|\vec{n_1} \cdot \vec{n_2}|}{|\vec{n_1}| |\vec{n_2}|}\)
    - 若为钝角（或求具体角）：\(\cos \theta = \frac{\vec{n_1} \cdot \vec{n_2}}{|\vec{n_1}| |\vec{n_2}|}\)
4.  **求正弦值：** 使用恒等式 \(\sin \theta = \sqrt{1 - \cos^2 \theta}\)。

**示例：**
已知一个二面角，你通过计算得到 \(\cos \theta = -\frac{1}{2}\)。

- 这说明 \(\theta\) 是钝角，具体是 \(120^\circ\)。
- 那么 \(\sin \theta = \sqrt{1 - (-\frac{1}{2})^2} = \sqrt{1 - \frac{1}{4}} = \sqrt{\frac{3}{4}} = \frac{\sqrt{3}}{2}\)。
- 你**不会**用 \(|\cos \theta| = \frac{1}{2}\) 作为答案，因为那是错的。

希望这个针对高中数学的解释能帮助您彻底理解这个问题！

# 3

立体几何中, 什么时候 sin = abs(cos) 什么时候 sin = sqrt(1-cos^2). 其中 sin 表示原角, cos 表示 (法) 向量的夹角.

---

在立体几何中，使用向量方法求角时，正弦值的计算方式取决于所求角的类型。具体如下：

### 1. 求直线与平面所成角（线面角）的正弦值

- 设直线的方向向量为 \(\vec{s}\)，平面的法向量为 \(\vec{n}\)，方向向量与法向量的夹角为 \(\phi\)。
- 线面角 \(\theta\) 满足关系：\(\sin \theta = |\cos \phi|\)。
- 推导：线面角 \(\theta\) 是直线与它在平面上的投影所成的角，有 \(\theta = 90^\circ - \phi\) 或 \(\theta = \phi - 90^\circ\)，故 \(\sin \theta = |\cos \phi|\)。
- 公式：\(\sin \theta = \dfrac{|\vec{s} \cdot \vec{n}|}{|\vec{s}| |\vec{n}|}\).

### 2. 求二面角的正弦值

- 设两个平面的法向量分别为 \(\vec{n_1}\) 和 \(\vec{n_2}\)，法向量的夹角为 \(\beta\)。
- 二面角 \(\alpha\) 满足关系：\(\sin \alpha = \sqrt{1 - \cos^2 \beta}\)。
- 推导：二面角 \(\alpha\) 与法向量夹角 \(\beta\) 满足 \(\beta = \alpha\) 或 \(\beta = 180^\circ - \alpha\)，故 \(\sin \alpha = \sin \beta\)。由于 \(\sin \beta = \sqrt{1 - \cos^2 \beta}\)，且 \(\sin \beta \geq 0\)，因此直接使用该公式。
- 公式：先求 \(\cos \beta = \dfrac{\vec{n_1} \cdot \vec{n_2}}{|\vec{n_1}| |\vec{n_2}|}\)，则 \(\sin \alpha = \sqrt{1 - \cos^2 \beta}\).

### 总结

- 当求**线面角**的正弦时，使用 \(\sin = |\cos|\)，其中 \(\cos\) 是方向向量与法向量夹角的余弦。
- 当求**二面角**的正弦时，使用 \(\sin = \sqrt{1 - \cos^2}\)，其中 \(\cos\) 是法向量夹角的余弦。

介绍 叉积

## 叉积（向量积）介绍

### 定义

叉积（又称向量积、外积）是向量代数中的一种二元运算，用符号 `×` 表示。它仅适用于三维空间中的向量。

对于两个三维向量 **a** = (a₁, a₂, a₃) 和 **b** = (b₁, b₂, b₃)，它们的叉积定义为：

```
a × b = (a₂b₃ - a₃b₂, a₃b₁ - a₁b₃, a₁b₂ - a₂b₁)
```

### 几何意义

#### 1. 方向

叉积结果是一个向量，其方向由**右手定则**确定：

- 右手四指从 **a** 弯向 **b**
- 大拇指指向即为 **a × b** 的方向

#### 2. 模长

叉积的模长等于以 **a** 和 **b** 为邻边的平行四边形的面积：

```
|a × b| = |a|·|b|·sinθ
```

其中 θ 是 **a** 与 **b** 之间的夹角。

### 重要性质

1. **反交换律**：a × b = - (b × a)
2. **与标量乘法兼容**：(k·a) × b = k·(a × b)
3. **分配律**：a × (b + c) = a × b + a × c
4. **与自身叉积为零**：a × a = 0
5. **与点积的关系**：a · (b × c) = (a × b) · c

### 应用

#### 1. 求法向量

叉积可用于求平面的法向量。如果平面由两个不平行向量 **u** 和 **v** 确定，则法向量为：

```
n = u × v
```

#### 2. 计算面积

- 平行四边形面积：S = |a × b|
- 三角形面积：S = ½|a × b|

#### 3. 判断向量关系

- 如果 a × b = 0，则 **a** 与 **b** 平行
- 如果 a × b ≠ 0，则 **a** 与 **b** 不平行

#### 4. 在立体几何中的应用

- 求二面角的正弦值：sinφ = |n₁ × n₂|/(|n₁|·|n₂|)
- 判断点与平面的位置关系
- 计算点到直线的距离

### 记忆技巧

叉积的计算公式可用行列式记忆：

```
     | i   j   k  |
a×b = | a₁  a₂  a₃ | = (a₂b₃-a₃b₂)i - (a₁b₃-a₃b₁)j + (a₁b₂-a₂b₁)k
     | b₁  b₂  b₃ |
```

其中 i, j, k 是坐标轴单位向量。

### 注意事项

1. 叉积仅定义在三维空间中
2. 结果是一个向量，不是标量
3. 不满足交换律，顺序很重要
4. 与点积不同，点积结果是标量，叉积结果是向量

叉积是解决三维几何问题的强大工具，特别在计算法向量、面积和判断方向时非常有用。

## 行列式介绍

### 定义

行列式是一个可以从方阵（行数和列数相等的矩阵）中计算出来的特殊数值。它是线性代数中的基本概念，用两条竖线表示，如 `|A|` 或 `det(A)`。

### 二阶行列式

对于 2×2 矩阵：

```
| a  b |
| c  d |
```

其行列式为：

```
det = a·d - b·c
```

### 三阶行列式

对于 3×3 矩阵：

```
| a  b  c |
| d  e  f |
| g  h  i |
```

其行列式可用**萨吕法则**计算：

```
det = a(e·i - f·h) - b(d·i - f·g) + c(d·h - e·g)
```

### 高阶行列式

n 阶行列式可以通过**拉普拉斯展开**（按行或列展开）来计算：

- 选择一行或一列
- 对每个元素，去掉该元素所在的行和列，得到(n-1)阶子式
- 带上符号(-1)^(i+j)（i,j 为元素位置）
- 求和：Σ[元素 × (-1)^(i+j) × 子式]

### 行列式的性质

#### 1. 基本性质

- **单位矩阵**：|I| = 1
- **零矩阵**：如果某行或某列全为 0，则行列式为 0
- **转置不变**：|A| = |Aᵀ|
- **行交换**：交换两行，行列式变号

#### 2. 线性性质

- **数乘**：某行乘以 k，行列式变为 k 倍
- **行加法**：某行加上另一行的倍数，行列式不变

#### 3. 乘法性质

- |A·B| = |A|·|B|
- |A⁻¹| = 1/|A|（当 A 可逆时）

### 几何意义

#### 二维情况

二阶行列式的绝对值表示以两个行向量为邻边的**平行四边形面积**。

#### 三维情况

三阶行列式的绝对值表示以三个行向量为棱的**平行六面体体积**。

#### n 维情况

n 阶行列式的绝对值表示 n 个 n 维向量张成的**n 维平行多面体的体积**。

### 计算方法

#### 1. 对角线法（仅适用于 2、3 阶）

- 2 阶：主对角线乘积减去副对角线乘积
- 3 阶：萨吕法则

#### 2. 高斯消元法

通过行变换将矩阵化为上三角矩阵，然后对角线元素相乘。

#### 3. 拉普拉斯展开

按某行或某列展开，化为低阶行列式计算。

### 重要应用

#### 1. 判断矩阵可逆性

- 矩阵可逆 ⇔ 行列式 ≠ 0
- 行列式 = 0 表示矩阵奇异（不可逆）

#### 2. 解线性方程组（克莱姆法则）

对于方程组 Ax = b，解为：

```
xᵢ = |Aᵢ|/|A|
```

其中 Aᵢ 是将 A 的第 i 列替换为 b 得到的矩阵。

#### 3. 计算特征值

特征多项式：det(A - λI) = 0

#### 4. 在向量代数中

- 叉积可用行列式表示：

```
a × b = | i   j   k  |
        | a₁  a₂  a₃ |
        | b₁  b₂  b₃ |
```

- 混合积（标量三重积）：a·(b×c) = det[a b c]

#### 5. 坐标变换

雅可比行列式表示坐标变换时的体积缩放因子。

### 特殊行列式

#### 1. 范德蒙行列式

```
| 1    1    ...  1   |
| x₁   x₂   ...  xₙ  |
| x₁²  x₂²  ...  xₙ² |
| ...  ...  ...  ... |
```

其值 = Π(xᵢ - xⱼ) (i > j)

#### 2. 对角行列式

等于对角线元素乘积。

#### 3. 三角行列式

等于主对角线元素乘积。

### 记忆技巧

- 2 阶："主减副"
- 3 阶：萨吕法则的"+-+-"符号规律
- n 阶：拉普拉斯展开时注意符号(-1)^(i+j)

### 注意事项

1. 行列式仅对**方阵**定义
2. 行列式是一个**数值**，不是矩阵
3. 不同阶数的行列式计算方法不同
4. 计算时注意符号变化

行列式是线性代数中的核心概念，它连接了矩阵理论与几何直观，在数学、物理、工程等领域都有广泛应用。
