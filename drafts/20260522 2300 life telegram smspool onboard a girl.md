smspool history

To view your SMSPool order log and re-access previously used phone numbers, you need to check your Order History. \[[1](https://www.smspool.net/article/how-to-access-your-smspool-order-history-1313b661ba99)]

How to Access Order History on the Web

1. Log in to your [SMSPool Account Dashboard](https://www.smspool.net/).
2. Click the **Settings cog icon** located in the top right corner of the interface.
3. Select **Order History** from the dropdown menu.
4. Alternatively, bypass the navigation entirely by using the direct [SMSPool Order History Page](https://www.smspool.net/article/how-to-access-your-smspool-order-history-1313b661ba99). \[[1](https://www.smspool.net/article/how-to-access-your-smspool-order-history-1313b661ba99)]

Managing History via the API

Developers can programmatically query past SMS requests via the SMSPool API Docs. \[[1](https://documenter.getpostman.com/view/30155063/2s9YXmZ1JY)]

- Send an **HTTP POST** request to `https://api.smspool.net/request/history`.
- Include your private `key` parameter (your API key).
- The server will return a **JSON array** containing details like the `order_code`, `phonenumber`, `code`, `cost`, and `timestamp`. \[[1](https://documenter.getpostman.com/view/30155063/2s9YXmZ1JY)]

Why Check Your History?

- **Unarchive Phone Numbers**: If a temporary number is no longer visible on your active "Order" dashboard, you can locate it here to bring it back.
- **Reactivate for New Codes**: Locate the phone number within the log, click **Reactivate**, and if successful, you can return to the main order page to send a new OTP code. Only completed orders qualify for reuse.
- **Audit Spending**: Provides a comprehensive historical tracking sheet of your logs, verification states, and pricing defaults. \[[1](https://www.smspool.net/article/how-to-reactivate-an-unused-phone-number-5fdb1b638523), [2](https://www.smspool.net/article/how-to-reuse-a-phone-number-on-smspool-94962c35b3), [3](https://www.smspool.net/article/how-to-access-your-smspool-order-history-1313b661ba99)]

can i use that again?

**Yes, you can reuse an SMSPool phone number**, but only under certain conditions. SMSPool calls this process **Reactivation**.

Rules for Reusing Numbers

- **Short-Term Rentals Only**: You can only reuse numbers purchased through the "Quick SMS" (short-term) service.
- **Successful Orders Only**: The number must have successfully received a code during its initial rental. Failed orders cannot be reused.
- **Availability Dependable**: The physical SIM card must still be online and active in the provider's modem pool. If the carrier disconnected it, it is gone forever.
- **Extra Cost**: Every reactivation costs the same price as buying a brand-new number.

How to Reuse the Number

1. Navigate to your **Order History** page.
2. Find the specific phone number in your log.
3. Click the **Reactivate** button next to it.
4. If successful, the number moves back to your active dashboard.
5. Go to your active dashboard and click **Check for SMS** to wait for your new code.

If the reactivation fails, it means the number is permanently offline, and you must purchase a new one.

no Reactivate button

If there is **no Reactivate button** next to the phone number in your SMSPool order log, it means the number is **permanently gone and cannot be used again**.

The button disappears automatically for any of the following reasons:

Why the Button is Missing

- **The Number went Offline**: The physical SIM card has been disconnected, powered down, or recycled by the mobile provider's hardware pool.
- **The Order Failed Initially**: If the number never received a text message code during the original rental window, it is considered expired and cannot be reactivated.
- **Time Has Run Out**: Temporary numbers usually only stay in the system for **24 hours** (though some US pools like Foxtrot can last 3 to 5 days). Once that timeframe passes, the system completely purges the number from active hardware. \[[1](https://www.smspool.net/article/how-to-reactivate-an-unused-phone-number-5fdb1b638523), [2](https://www.justanswer.com/phone-systems/mzawe-old-phone-number-allowed-go-inactive.html), [3](https://www.smspool.net/article/how-to-reuse-a-phone-number-on-smspool-94962c35b3), [4](https://www.smspool.net/article/frequently-asked-questions-66559ffe55)]

What You Can Do Now

- **Buy a New Temporary Number**: Go back to the [SMSPool Dashboard](https://www.smspool.net/) and place a fresh order for your service. You will have to update the phone number on whatever app or website you are trying to log into.
- **Look Into Long-Term Rentals**: If you need a number that you can access repeatedly without it disappearing, use the **Long-term order** tab on the SMSPool side menu. These give you dedicated access for a full month at a time. \[[1](https://www.smspool.net/article/frequently-asked-questions-66559ffe55)]
