Here's a categorized list of JavaScript array methods:

### **Mutating Methods** (Modify the original array)

- `push()` - Adds elements to the end
- `pop()` - Removes last element
- `unshift()` - Adds elements to the beginning
- `shift()` - Removes first element
- `splice()` - Adds/removes elements at specific positions
- `reverse()` - Reverses array order
- `sort()` - Sorts array elements
- `fill()` - Fills elements with static value
- `copyWithin()` - Copies elements within the array

### **Non-Mutating Methods** (Return new array/value)

- `concat()` - Merges arrays
- `join()` - Creates string from array
- `slice()` - Extracts section of array
- `indexOf()` - Returns first index of element
- `lastIndexOf()` - Returns last index of element
- `includes()` - Checks if value exists
- `toString()` - Converts to string
- `toLocaleString()` - Localized string conversion

### **Iteration Methods**

- `forEach()` - Executes function for each element
- `map()` - Creates new array with results of function
- `filter()` - Creates filtered array
- `reduce()` - Reduces to single value (left-to-right)
- `reduceRight()` - Reduces (right-to-left)
- `every()` - Tests if all elements pass test
- `some()` - Tests if any element passes test
- `find()` - Returns first matching element
- `findIndex()` - Returns index of first match
- `findLast()` - Returns last matching element (ES2023)
- `findLastIndex()` - Returns index of last match (ES2023)
- `flatMap()` - Maps then flattens result
- `keys()` - Returns iterator of keys
- `values()` - Returns iterator of values
- `entries()` - Returns iterator of key/value pairs

### **Array Transformation**

- `flat()` - Creates flattened array
- `toReversed()` - Returns reversed copy (ES2023)
- `toSorted()` - Returns sorted copy (ES2023)
- `toSpliced()` - Returns spliced copy (ES2023)
- `with()` - Returns copy with element changed (ES2023)

### **Static Methods** (Called on Array constructor)

- `Array.isArray()` - Checks if value is array
- `Array.from()` - Creates from array-like/iterable
- `Array.of()` - Creates from arguments

### **ES6+ Methods** (Modern JavaScript additions)

- `includes()` (ES7)
- `flat()` (ES2019)
- `flatMap()` (ES2019)
- `findLast()` (ES2023)
- `findLastIndex()` (ES2023)
- `toReversed()` (ES2023)
- `toSorted()` (ES2023)
- `toSpliced()` (ES2023)
- `with()` (ES2023)

This categorization helps understand:

- Which methods modify the original array vs return new values
- Methods for searching and filtering
- Methods for transformation and iteration
- Modern ES6+ additions for more functional programming patterns
