# adb command to export all apks from an android phone on ubuntu

To export all APK files from an Android phone to your Ubuntu machine, you must use a terminal script because `adb` does not have a single native command to pull all apps at once.

### 🔧 Prerequisites

- Install ADB on Ubuntu: `sudo apt install adb`
- Enable **USB Debugging** on your phone.
- Connect the phone via USB.
- Verify connection with: `adb devices` \[[1](https://developer.android.com/develop/devices/chromeos/learn/development-environment), [2](https://medium.com/@nutts.786/how-to-take-screenshot-and-screen-record-via-adb-e7d1fa75d02e), [3](https://www.debugpoint.com/how-to-access-android-devices-internal-storage-and-sd-card-in-ubuntu-linux-mint-using-media-transfer-protocol-mtp/), [4](https://www.yahoo.com/tech/top-adb-commands-every-android-200014787.html)]

### 💻 The Bash Script Command

Run this exact command in your Ubuntu terminal. It automatically loops through every installed app and pulls the APK files into a new folder named `extracted_apks` in your current directory: \[[1](https://medium.com/@silverskytechnology/creating-apks-from-aab-using-bundletool-644477a2f096), [2](https://www.repeato.app/efficiently-installing-apks-on-multiple-connected-android-devices/)]

bash

```
mkdir -p extracted_apks && adb shell pm list packages -f | sed 's/package://g' | while read -r line; do apk="${line%%=*}"; pkg="${line##*=}"; echo "Pulling $pkg..."; adb pull "$apk" "./extracted_apks/${pkg}.apk"; done
```

Use code with caution.

### 🔍 How It Works

- `mkdir -p extracted_apks`: Creates the target storage folder.
- `adb shell pm list packages -f`: Lists all app file paths and package names.
- `sed 's/package://g'`: Cleans the terminal text output.
- `while read... adb pull`: Extracts each APK and renames it to its actual package name. \[[1](https://wiki.t-firefly.com/ROC-RK3328-CC/android%5Fadb.html)]

### 🎯 Alternative Filters

If you do not want _every_ app, modify the script by changing the `adb shell pm list packages` part:

- **Third-party apps only**: Change to `adb shell pm list packages -3 -f`
- **System apps only**: Change to `adb shell pm list packages -s -f`
- **Specific app search**: Change to `adb shell pm list packages -f | grep "google"`
