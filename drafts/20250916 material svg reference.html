SVG Reference Guide

Scalable Vector Graphics - Examples and Documentation

SVG is an XML-based vector image format for defining two-dimensional graphics. SVG images can be scaled indefinitely without losing quality and are perfect for responsive web design.

Basic Shapes

SVG provides several basic shape elements for drawing common shapes.

Rectangle

<rect x="20" y="20" width="100" height="80" fill="#3498db" />

Circle

<circle cx="100" cy="100" r="60" fill="#e74c3c" />

Polygon

<polygon points="100,20 40,180 190,80 10,80 160,180" fill="#9b59b6" />

Paths

The <path> element is used to create complex shapes by combining lines, curves, and arcs.

Curved Path

<path d="M 30 100 Q 150 20 270 100 T 500 100" stroke="#2c3e50" fill="none" />

Custom Shape

<path d="M100,50 C150,150 50,150 100,250 C150,150 250,150 200,50 Z" fill="#2ecc71" />

Text

SVG allows you to render text directly within graphics.

Basic Text

<text x="50%" y="50%" text-anchor="middle" font-size="24">SVG Text Example</text>

Text on Path

<path id="textPath" d="M50,150 Q150,50 250,150" fill="none"/>
<text>
  <textPath href="#textPath">Text on path</textPath>
</text>

Gradients & Patterns

SVG supports gradient fills and pattern definitions for more complex styling.

Linear Gradient

<linearGradient id="grad1" x1="0%" y1="0%" x2="100%" y2="0%">
  <stop offset="0%" stop-color="#3498db" />
  <stop offset="100%" stop-color="#2ecc71" />
</linearGradient>
<rect fill="url(#grad1)" ... />

Radial Gradient

<radialGradient id="grad2" cx="50%" cy="50%" r="50%">
  <stop offset="0%" stop-color="#ffffff" />
  <stop offset="100%" stop-color="#e74c3c" />
</radialGradient>
<circle fill="url(#grad2)" ... />

Filters & Effects

SVG filters allow you to apply sophisticated effects to your graphics.

Blur Effect

<filter id="blur">
  <feGaussianBlur in="SourceGraphic" stdDeviation="5" />
</filter>
<circle filter="url(#blur)" ... />

Drop Shadow

<filter id="shadow" ... >
  <feGaussianBlur ... />
  <feOffset ... />
  <feMerge> ... </feMerge>
</filter>
<rect filter="url(#shadow)" ... />

Animation

SVG supports animation through SMIL or can be animated with CSS and JavaScript.

Animated Circle

<circle cx="100" cy="100" r="30" fill="#3498db">
  <animate attributeName="r" from="10" to="50" dur="2s" repeatCount="indefinite" />
</circle>

Motion Along Path

<path id="motionPath" d="M50,150 Q150,50 250,150" ... />
<circle r="10" fill="#e74c3c">
  <animateMotion dur="3s" repeatCount="indefinite">
    <mpath href="#motionPath"/>
  </animateMotion>
</circle>

Common SVG Properties

fill

Defines the color of the shape's interior

stroke

Defines the color of the shape's outline

stroke-width

Defines the thickness of the outline

opacity

Defines the transparency of the element (0-1)

transform

Applies transformations like rotate, scale, translate, skew

viewBox

Defines the position and dimension of the SVG viewport

SVG Usage Tips

Inline vs External

SVG can be used inline in HTML or as external files. Inline SVG is better for interaction and animation.

Accessibility

Use <title> and <desc> elements to provide accessible descriptions of your SVG content.

Performance

For complex SVGs, consider optimizing the code with tools like SVGO to reduce file size.
