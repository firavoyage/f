UnoCSS API Guide

The instant on-demand Atomic CSS engine

What is UnoCSS?

UnoCSS is an atomic CSS engine that provides instant utility classes without any configuration. It's highly customizable and works with any framework.

Key Features:
  • Fully customizable - no core utilities, all provided via presets
  • Instant on-demand CSS - no build time cost for unused utilities
  • Short, intuitive class names
  • Flexible enough to create any utility you need
  • Works with any framework (Vue, React, Svelte, etc.)

Core Concepts

Presets

UnoCSS uses presets to provide utility classes. The default preset includes utilities for:

  • Spacing (margin, padding)
  • Colors (text, background, border)
  • Typography (font size, weight, family)
  • Flexbox and Grid
  • And much more...

Customization

You can easily create your own utilities or extend existing ones:

// uno.config.js
import { defineConfig } from 'unocss'

export default defineConfig({
  rules: [
    ['my-rule', { color: 'red' }]
  ]
})

Variants

Variants allow you to apply utilities conditionally:

// Hover variant
<button class="bg-blue-500 hover:bg-blue-700">Button</button>

// Dark mode variant
<div class="bg-white dark:bg-gray-800">Theme aware</div>

// Responsive variants
<div class="w-full md:w-1/2 lg:w-1/3">Responsive width</div>

Installation Guide

Via CDN (Quick Start)

<script src="https://cdn.jsdelivr.net/npm/@unocss/runtime"></script>

npm Installation

npm install -D unocss

Framework Integrations

UnoCSS has dedicated integrations for popular frameworks:

  • Vite: @unocss/vite
  • Webpack: @unocss/webpack
  • Nuxt: @unocss/nuxt
  • React: Use with Vite or Webpack
