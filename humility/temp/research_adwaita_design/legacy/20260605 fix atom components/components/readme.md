# atoms

- `Button` triggers an action when clicked
- `Checkbox` allows user to select or deselect an option
- `Input` text field for entering a single line of text
- `Switch` toggle control for on/off binary settings
- `Slider` control for selecting a value by dragging a thumb
- `Select` dropdown menu for selecting one or more options
- `NumberField` input field for entering numeric values with increment/decrement
- `ScrollArea` container with scrollable content and styled scrollbars
- `Tooltip` popup that displays descriptive text on hover or focus
