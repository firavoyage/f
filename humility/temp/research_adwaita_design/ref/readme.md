# Reference Implementations

Research materials and reference implementations for Adwaita web port.

## libadwaita

Original GNOME GTK4 widget library (C).

- Source: `src/` - widget implementations
- Styles: `src/stylesheet/` - SCSS styles
- Docs: `doc/` - documentation (CSS variables, style classes, etc.)

## adwaita_web

Web toolkit by romgrk implementing Adwaita for web.

- Packages: `packages/web-toolkit/`
- Focus: Practical web port with button sizes

## adwave

Alternative CSS implementation by ncpa0.

- Focus: Component-level state tokens (hover, pressed, etc.)
- Docs: `docs/components/`

## gnome_hig

GNOME Human Interface Guidelines.

- Source patterns: `source/patterns/`
- Focus: Design patterns and principles
